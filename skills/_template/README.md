# AI Skill Template

Use this folder as a starting point for new tools in the AI skills library.

## Files

- `SKILL.md`: AI-facing instructions for understanding and using the tool.
- `logic.py`: Example Python entrypoint.
- `README.md`: Human-readable setup and usage guidance.

## Setup

1. Copy this entire folder into a new folder under `/skills`.
2. Update `SKILL.md` with the tool name, description, and how the AI should use it.
3. Replace `logic.py` with your actual implementation.
4. Add any required environment variables to the `Security` section of `SKILL.md`.

## Run

```bash
python logic.py
```
